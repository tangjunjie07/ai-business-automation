# apps/web 必須 Secrets 一覧

以下は CI / デプロイで利用する想定のシークレット名（リポジトリまたは組織のSecretsに登録してください）。

共通（両環境）
- `SENTRY_DSN_STAGING`, `SENTRY_DSN_PROD`
- `SENTRY_TRACES_SAMPLE_RATE_STAGING`, `SENTRY_TRACES_SAMPLE_RATE_PROD`

ステージング専用
- `DATABASE_URL_STAGING`
- `NEXTAUTH_URL_STAGING`
- `NEXTAUTH_SECRET_STAGING`
- `STORAGE_PROVIDER_STAGING`
- `AWS_S3_BUCKET_STAGING`
- `AWS_REGION_STAGING`
- `AWS_ACCESS_KEY_ID_STAGING`
- `AWS_SECRET_ACCESS_KEY_STAGING`
- `INGESTION_URL_STAGING`

プロダクション専用
- `DATABASE_URL_PROD`
- `NEXTAUTH_URL_PROD`
- `NEXTAUTH_SECRET_PROD`
- `STORAGE_PROVIDER_PROD`
- `AWS_S3_BUCKET_PROD`
- `AWS_REGION_PROD`
- `AWS_ACCESS_KEY_ID_PROD`
- `AWS_SECRET_ACCESS_KEY_PROD`
- `INGESTION_URL_PROD`

デプロイ/ランタイム用（必要に応じて）
- `DEPLOY_API_KEY`（CD/プラットフォーム向け）

備考
- Secrets 名はチームルールで統一してください。CIやCDのステップで正しい名前を参照する必要があります。
